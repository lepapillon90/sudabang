# 2. 심화 질문 프롬프트
> AI가 아이디어를 제안했을 때 더 깊이 파고드는 질문

---

## 💡 구현 방식 질문

```
"제가 생각한 기능을 넣는다면 어떤 방식으로 구현하는 게 좋을까?"
```

**더 구체적인 버전:**
```
"[특정 기능]을 구현하려면:
1. 어떤 기술 스택이 필요할까?
2. 백엔드는 어떻게 설계해야 할까?
3. 실시간 기능은 어떻게 처리해야 할까?"
```

---

## 🚀 차별화 포인트 질문

```
"기존 앱들과 차별화될 수 있는 독특한 기능은 없을까?"
```

**더 구체적인 버전:**
```
"[경쟁 앱들]과 비교했을 때:
1. 어떤 페인 포인트를 해결할 수 있을까?
2. 우리만의 unique selling point(USP)는 뭘까?
3. 사용자들이 우리 앱을 선택할 이유는 뭘까?"
```

---

## 🔄 피벗 가능성 탐색

```
"이 아이디어를 다른 방향으로 발전시킬 수 있을까?"
"핵심 가치는 유지하면서 타겟을 바꾸면 어떨까?"
"B2C 대신 B2B로 접근하면 어떨까?"
```
