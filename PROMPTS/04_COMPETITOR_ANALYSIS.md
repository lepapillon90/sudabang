# 4. 경쟁 앱 정보 요청
> 시장 분석과 차별화 전략을 위한 프롬프트

---

## 🔍 기본 경쟁 분석 요청

```
"관련 커뮤니티 앱이나 플랫폼에는 어떤 것들이 있어? 
각 서비스의 주요 특징과 장단점을 알려줘."
```

---

## 📊 상세 경쟁 분석 템플릿

```
"[카테고리] 분야의 경쟁 앱을 분석해줘:

분석할 내용:
- 앱 이름 및 기본 정보
- 핵심 기능 3가지
- 장점: 사용자들이 좋아하는 점
- 단점: 사용자들이 불만인 점
- 타겟 사용자: 누구를 위한 앱인지
- 수익 모델: 어떻게 돈을 버는지
- 우리가 배울 점 / 차별화할 점"
```

---

## 🆚 비교 분석 요청

```
"[경쟁 앱 A], [경쟁 앱 B], [경쟁 앱 C]를 비교해줘.
표로 정리해서 각 앱의 강점과 약점을 보여줘."
```

---

## 💡 차별화 전략 도출

```
"경쟁 분석 결과를 바탕으로:
1. 시장에서 비어있는 틈새(gap)는 뭐야?
2. 우리가 집중해야 할 차별화 포인트는?
3. 경쟁 앱들의 약점을 어떻게 공략할 수 있을까?"
```
