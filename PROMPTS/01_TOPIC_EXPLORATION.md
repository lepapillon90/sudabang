# 1. 주제 탐색 프롬프트
> 기본 아이디어를 제시하고 탐색하기

---

## 💬 기본 아이디어 제시하기

**프롬프트 예시:**
```
1. 친구들이랑 소통할 수 있는 기능을 가진 웹형식 SNS를 만들려고 해 
   (예를 들어 카카오톡)

2. 서로의 지식을 공유하거나 주식공부를 하거나 할 수 있는 
   자기계발을 할 수 있는 기능도 있으면 합니다.
```

---

## 🎯 효과적인 주제 탐색 구조

```
나는 [A]와 [B]를 결합한 앱을 만들고 싶어.

주요 기능:
1. [핵심 기능 1] - (예: 실시간 채팅)
2. [핵심 기능 2] - (예: 지식 공유)
3. [핵심 기능 3] - (예: 학습 트래킹)

타겟 사용자: [누구를 위한 앱인가?]
참고할 만한 앱: [카카오톡, 네이버 카페 등]
```

---

## 📝 추가 탐색 질문 예시

```
- 이런 앱이 시장에서 성공하려면 어떤 요소가 필요할까?
- 비슷한 서비스는 어떤 것들이 있어?
- 이 아이디어의 강점과 약점은 뭐야?
```
