# 9. 수익 모델 요청
> 앱의 수익화 전략을 설계하기 위한 프롬프트

---

## 💰 기본 수익 모델 요청

```
"우리 앱의 초기 수익모델 및 향후 진행해야 할 수익 모델을 정의해줘. 
현실적인 조건으로 진행해줘."
```

---

## 📊 상세 수익 모델 템플릿

```
"[프로젝트명]의 수익 모델을 단계별로 설계해줘:

1. 초기 단계 (0-6개월)
   - 무료 서비스로 사용자 확보
   - 수익 없음 또는 최소 수익

2. 성장 단계 (6-12개월)
   - 프리미엄 기능 도입
   - 구독 모델 또는 인앱 결제

3. 확장 단계 (12개월 이후)
   - 다양한 수익원 추가
   - B2B 또는 광고 모델

각 단계별 예상 수익과 필요한 사용자 수를 포함해줘."
```

---

## 🎯 수익 모델 유형 분석 요청

```
"우리 앱에 적용 가능한 수익 모델들을 분석해줘:

- 프리미엄(Freemium): 무료 + 유료 기능
- 구독(Subscription): 월/연 정기 결제
- 인앱 결제(In-App Purchase): 아이템/기능 구매
- 광고(Advertising): 배너/리워드 광고
- 커미션(Commission): 거래 수수료
- B2B 라이선스: 기업용 판매

각 모델의 장단점과 우리 앱에 적합한 이유를 설명해줘."
```

---

## 💵 현실적인 수익 예측 요청

```
"다음 조건에서 예상 수익을 계산해줘:

- 월간 활성 사용자(MAU): [X]명
- 유료 전환율: [Y]%
- 평균 객단가(ARPU): [Z]원
- 리텐션율: [W]%

1년 후 예상 월 매출은?"
```

---

## 🔄 수익 모델 비교 요청

```
"비슷한 앱들의 수익 모델을 조사해줘:
- [경쟁 앱 A]의 수익 모델
- [경쟁 앱 B]의 수익 모델

우리가 참고할 수 있는 포인트와 
차별화할 수 있는 기회는?"
```

---

## 💡 수익 모델 심화 질문

```
"초기에 무료로 시작한다면, 언제 유료화를 도입하는 게 좋을까?"
"사용자 이탈 없이 유료화하는 전략은?"
"우리 앱의 LTV(고객 생애 가치)를 높이는 방법은?"
```

---

## 📋 가격 책정 요청

```
"프리미엄 구독 요금을 책정해줘:

고려사항:
- 타겟 사용자의 지불 의향
- 경쟁 앱의 가격대
- 제공되는 가치
- 시장 포지셔닝

월간/연간 구독 가격과 그 근거를 설명해줘."
```
