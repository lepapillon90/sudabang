# 📱 수다방 (SudaBang) - PRD

> 혼자보다 함께, 수다처럼 편하게 성장하는 자기계발 커뮤니티

---

## 1. 프로젝트 개요

### 1.1 프로젝트 배경
- 혼자 공부/자기계발하면 동기부여가 어려움
- 기존 SNS는 오락 중심, 기존 커뮤니티는 실시간 소통 부족
- **함께 성장**하는 문화에 대한 니즈 증가

### 1.2 핵심 가치
**"수다처럼 편하게, 함께 성장하는 커뮤니티"**

### 1.3 타겟 사용자
| 페르소나 | 특징 |
|----------|------|
| 김지훈 (25세, 취준생) | 주식/자기계발 관심, 동기부여 필요, 공부 인증 원함 |
| 이수진 (30세, 직장인) | 퇴근 후 사이드 프로젝트, 전문가 조언 필요 |

---

## 2. 핵심 기능 (MVP)

### P0 - 필수 기능
| 기능 | 설명 |
|------|------|
| 🔐 회원가입/로그인 | Google/이메일 로그인 (Firebase Auth) |
| 💬 관심사별 채팅방 | 주식, 독서, 운동, 개발, 언어 등 카테고리별 그룹 |
| 🎯 목표 설정 | 개인 목표 설정 및 달성률 시각화 |
| 👤 프로필 | 관심사, 목표, 활동 내역 |

### P1 - 있으면 좋은 기능
| 기능 | 설명 |
|------|------|
| 📝 피드 | 오늘의 공부/활동 인증 게시글 |
| ❤️ 좋아요/댓글 | 피드 상호작용 |
| 🔔 알림 | 채팅, 댓글 알림 |

### P2 - 향후 추가 기능
| 기능 | 설명 |
|------|------|
| 🏆 챌린지 | 21일 습관 만들기 등 |
| 👨‍🏫 멘토-멘티 | 경험자-초보자 매칭 |
| 📚 자료실 | 학습 자료 공유 라이브러리 |

## 3. 기술 스택

| 영역 | 기술 | 선택 이유 |
|------|------|----------|
| **Frontend** | Next.js 14 (App Router) | SSR, 최신 React 기능, 빠른 개발 |
| **Language** | TypeScript | 타입 안정성, 유지보수성 |
| **Styling** | Tailwind CSS | 빠른 스타일링, 일관된 디자인 |
| **Backend** | Firebase | 빠른 구축, 실시간 DB, 인증 |
| **Database** | Firestore | NoSQL, 실시간 동기화 |
| **Storage** | Firebase Storage | 이미지 업로드 |
| **Auth** | Firebase Auth | Google, 이메일 로그인 |
| **Deployment** | Vercel | Next.js 최적화, 무료 티어 |

## 4. 화면 구성

```
/                   → 랜딩 페이지
/login              → 로그인
/signup             → 회원가입
/onboarding         → 관심사 선택
/home               → 메인 (피드)
/rooms              → 채팅방 목록
/rooms/[id]         → 채팅방 상세
/goals              → 내 목표
/profile            → 프로필
/settings           → 설정
```
