# 🗺️ Feature Roadmap

## Phase 1: Stability & Core Experience (Current)
- [x] **운영 기반**: 사용자 인증(Auth), 채팅방 생성/입장, 피드 기능
- [x] **UX/UI**: 다크 모드 기반의 프리미엄 디자인, 반응형 레이아웃
- [x] **데이터**: Firestore 연동, 실시간 업데이트
- [ ] **알림**: 앱 내 기본적인 토스트 알림 (채팅, 목표 달성)

## Phase 2: Social & Interaction (Next Step)
- [ ] **실시간 알림 센터**: 
    - 누군가 내 글에 좋아요/댓글을 남겼을 때 알림
    - 채팅방 멘션(@) 기능
- [ ] **소셜 기능 강화**:
    - **팔로우/팔로워**: 관심 있는 멤버 구독하기
    - **1:1 DM**: 프라이빗한 대화 요청
- [ ] **피드 고도화**:
    - 이미지 다중 업로드
    - 해시태그(#) 지원 및 검색

## Phase 3: Gamification & Motivation
- [ ] **성장 시스템**:
    - 활동량(글, 댓글, 목표 달성)에 따른 '성장 점수' 부여
    - 등급 시스템 (예: Seed -> Sprout -> Tree -> Forest)
- [ ] **디지털 뱃지**:
    - "30일 연속 목표 달성", "100명의 파트너" 등 업적 뱃지
- [ ] **목표 챌린지**:
    - 다함께 참여하는 기간 한정 챌린지 (예: 미라클 모닝 2주)

## Phase 4: Monetization & Pro Feature
- [ ] **SudaBang Pro (멤버십)**:
    - 무제한 목표 설정
    - Pro 전용 프리미엄 테마
    - 통계 분석 리포트 (월간 성취도 분석)
- [ ] **Expert Lounge**:
    - 인증된 전문가가 운영하는 유료/프리미엄 세션 룸

## Phase 5: AI Integration
- [ ] **AI 회고 파트너**: 하루의 기록을 분석하여 회고 질문 던져주기
- [ ] **스마트 매칭**: 나와 성향이 비슷한 성장 메이트 추천
